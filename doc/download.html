Download instructions

The code is maintained in CVS.

The code repository head is
:pserver:anonymous@cdcvs.fnal.gov:/cvs/cd_read_only
and the package name is
glideinWMS

So the complete BASH command would be: 
cvs -d :pserver:anonymous@cdcvs.fnal.gov:/cvs/cd_read_only co -r v2_4_3 glideinWMS

The package can also be downloaded as a tarball:
  • Current Production Series: glideinWMS_v2_4_3.tgz
  • Previous Production Series: glideinWMS_v1_6_3.tgz

Special instructions for v1_6_3 and v2_3_2

v1_6_3 and v2_3_2 contain security fixes. These releases fixes the way certificate subjects are mapped in condor_mapfile. If you are upgrading to these releases over the existing releases, you need to manually apply the fix to condor_mapfile used by condor in your installation. 

 Usage:
 convert_condormap infile outfile

After converting the condor_mapfile, please make sure that all the entries have been written back into the convered mapfile
